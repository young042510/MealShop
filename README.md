# 🖥️ 밀키트 판매 사이트

SPRINGBoot를 이용한 기본적인 판매 사이트 구현

## 구성원 & 역할


이**(나) - 회원가입/ 로그인/ 상품문의/ 장바구니

송** (팀원) - 관리자/ 이벤트/ 상품 및 리뷰/ 찜 /주문
